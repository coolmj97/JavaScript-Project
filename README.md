# JavaScript-Project
## 1. 가계부 (account book)
* 시맨틱 마크업   
* 반응형 레이아웃   

* 기능   
  - form 제출 시 리스트 생성 및 로컬 스토리지 저장   
  - 리스트에 hover시 수정, 삭제 버튼   
  - 수입/지출 필터   
  - 낮은 금액순, 높은 금액순, 과거순, 최근순 정렬   
  - 파싱   
  - 빈 폼을 제출시도하면 제출을 막기   
  
* 기술 스택   
  - HTML   
  - CSS   
  - Bootstrap   
  - es6 JavaScript      
  
* 링크 : [바로가기](https://coolmj97.github.io/JavaScript-Project/account-book/)
